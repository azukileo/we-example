# We.js example and skeleton project

> Live example: http://wejs.org/

## Instalation

```sh
// clone the skeleton
$ git clone https://github.com/wejs/we-example.git we-project
// enter in folder
$ cd we-project
// install depencencies
$ npm install && bower install
```
